# Formatação DOCX para Representações Cautelares

Use este padrão para gerar peças cautelares em `.docx` com apresentação profissional, reaproveitando o padrão visual adotado na skill `relatorio-final-ip`.

## Configurações do documento

- Página A4
- Margens:
  - superior: 1440
  - inferior: 1440
  - esquerda: 1800
  - direita: 1080
- Fonte do corpo: Arial 12 pt
- Espaçamento entre linhas: 1,5
- Alinhamento padrão: justificado

## Hierarquia visual

- Título principal: Arial 16 pt, negrito, caixa alta, centralizado
- Títulos de seção: Arial 14 pt, negrito, justificado
- Subtítulos e itens: Arial 12 pt, negrito
- Corpo: Arial 12 pt, justificado

## Cabeçalho

Em regra, usar duas linhas:

1. POLÍCIA CIVIL DO ESTADO OU ÓRGÃO CORRESPONDENTE
2. NOME DA UNIDADE POLICIAL

Se o caso concreto exigir, admitir terceira linha com setor ou delegacia especializada.

## Rodapé

- Identificação do procedimento, preferencialmente `IP nº _____`
- Numeração de páginas, quando tecnicamente disponível

## Regras práticas

- Não entregar apenas Markdown quando a peça já estiver madura para protocolo ou revisão final.
- Na ausência de restrição expressa do usuário, gerar também o `.docx` final.
- Se houver minuta em Markdown, usá-la como fonte para o `.docx`, preservando títulos, listas e destaques.
- Confirmar ao final o caminho do arquivo gerado.
